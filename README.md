# Sales Trend Forecaster 📈

Link: [Sales Trend Forecaster](https://dl-project03-sales-forecaster.streamlit.app/)

A Deep Learning project built with TensorFlow/Keras and Streamlit. This project uses a **Long Short-Term Memory (LSTM)** neural network to analyze time-series data and predict future daily sales volume.

## Features
* Custom LSTM Deep Learning Architecture.
* Time-series data sequencing and scaling (`MinMaxScaler`).
* Interactive Web UI using Streamlit to visualize past sales vs. AI predictions.
* Trained using Mean Squared Error (`mean_squared_error`) loss.

## Tech Stack
* Python
* TensorFlow / Keras
* Streamlit
* Pandas & NumPy
* Scikit-Learn
* Matplotlib
